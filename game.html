❓ Game Help & Pro Tips

RGB Matching Game Tips

  • Easy: 3 options, 60s | +10 points/correct | Insane: 12 options, 20s | +30 points/correct
  • Combo bonus: +2 points per consecutive correct answer, resets on wrong answer
  • Wrong answer deducts 5 points, level up every 5 correct answers
  • Time bonus: +1 point per remaining second when answering correctly

Hex Guess Game Tips

  • Hex codes are case-insensitive, # is auto-completed for you
  • Hint system: reveals 1 digit per use, but deducts 5 points from correct answer
  • Timed mode: 60 seconds to guess as many colors as possible
  • Short hex codes (e.g., #F00) are accepted as valid answers

CMYK Challenge Tips

  • Real-time match accuracy shows how close you are to the target
  • 5% tolerance for perfect match, 10% tolerance for partial match
  • Successful matches give 20 points + accuracy bonus
  • K (Black) is the key to darkening colors in print design

HSL Master Tips

  • Hue: 0°=Red, 120°=Green, 240°=Blue, 360°=Red (full color wheel)
  • Saturation: 0% = Grayscale, 100% = Full vivid color
  • Lightness: 0% = Black, 50% = True Color, 100% = White
  • HSL is the most intuitive model for adjusting shade and tint
