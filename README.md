## Project Description
This project is the coursework for EBU6305. We develop a learning-oriented web application around Color Coding, including 5 core functional pages and 1 support page. The application is compatible with multi-terminal browsers, integrates interactive design and game elements, and focuses on the popularization and practical application of color coding knowledge. It supports inclusive design standards such as bilingual switching (English/Chinese) and adaptive display for different screen sizes, and incorporates sound design to enhance user experience.
