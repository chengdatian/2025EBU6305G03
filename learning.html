RGB Color Model | Additive Light System

RGB (Red, Green, Blue) is an additive color model where red, green, and blue light are added together in various ways to reproduce a broad array of colors. Each channel ranges from 0 (no light) to 255 (full intensity).

🔑 Core Term

Additive Color: Light sources are combined to create new colors

📊 Color Depth

24-bit True Color → 16,777,216 unique colors

📚 Key Knowledge Points (Expanded)

  • Used for electronic screens (monitors, phones, TVs, projectors) - additive color mixing
  • Each channel (R/G/B) has 256 levels (0-255) → 16,777,216 possible colors
  • Common RGB values: White (255,255,255), Black (0,0,0), Pure Red (255,0,0), Blue (0,0,255)
  • Web development: Used in CSS (rgb() / rgba() with alpha transparency channel)
  • Digital imaging: Photoshop, Figma, Canva default color mode for screen design
  • RGBA: Adds transparency (0 = fully transparent, 1 = fully opaque)

💼 Industry Applications

Web Design · Mobile Apps · Game Development · Digital Advertising · UI/UX Design · Social Media Graphics

CMYK Color Model | Subtractive Printing System

CMYK (Cyan, Magenta, Yellow, Key Black) is a subtractive color model used in color printing. It works by subtracting (absorbing) light wavelengths and reflecting others.

🔑 Core Term

Subtractive Color: Ink absorbs light, creating color

🖨️ Key Fact

K = Key Black → prevents muddy dark brown from CMY mix

📚 Key Knowledge Points (Expanded)

  • Used for physical printing (posters, magazines, business cards, books) - subtractive color mixing
  • CMY alone can't produce true black (results in dark brown) → Key Black (K) is added
  • 100% C + 100% M + 100% Y + 0% K = Rich Black (K used for sharp text & deep blacks)
  • Printing resolution: DPI (Dots Per Inch) - standard is 300 DPI for high-quality prints
  • Professional software: Adobe Illustrator, InDesign, Photoshop (Print Mode)
  • Color calibration: Critical for matching screen colors to printed results

💼 Industry Applications

Offset Printing · Packaging · Brand Materials · Magazines · Business Cards · Large Format Posters

Hexadecimal Code | Web Standard Color Format

Hexadecimal color codes are a shorthand for RGB values, represented as a 6-digit combination of numbers 0-9 and letters A-F (0-15 in decimal). The format is #RRGGBB.

🔑 Base System

Hexadecimal = Base 16 (0-9, A-F)

🌐 Web Standard

Official HTML/CSS color code since 1996

🎨 Interactive Hex Color Converter

Corresponding RGB Value: rgb(255, 0, 0)

Short Hex (if applicable): #F00

Corresponding HSL Value: hsl(0, 100%, 50%)

📚 Key Knowledge Points (Expanded)

  • Most common color format in front-end development (CSS, HTML, SVG)
  • Short hex code (#RGB) is a shorthand for #RRGGBB (e.g., #F00 = #FF0000)
  • Alpha channel (transparency) can be added: #RRGGBBAA (CSS RGBA equivalent)
  • Hex values are case-insensitive (e.g., #ff0000 = #FF0000 = #f00)
  • 8-digit Hex: Supports transparency (00 = transparent, FF = opaque)
  • Used in all design tools: Figma, Sketch, Adobe XD, Webflow

💼 Industry Applications

Web Development · UI Design · CSS Coding · SVG Graphics · Digital Design · App Development

HSL/HSV Color Model | Human-Friendly Color System

HSL (Hue, Saturation, Lightness) and HSV (Hue, Saturation, Value) are human-centric color models, designed to align with how humans perceive and describe colors. They are widely used in design tools and CSS for intuitive color adjustment.

🔑 Core Term

Hue: The base color, 0-360° on the color wheel

🎨 Human-Friendly

Adjust brightness/saturation without changing base hue

📚 Key Knowledge Points

  • Hue: 0° = Red, 120° = Green, 240° = Blue, 360° = Red (full color wheel)
  • Saturation: 0% = Grayscale, 100% = Full vivid color
  • Lightness: 0% = Pure Black, 50% = True Color, 100% = Pure White
  • HSV is similar to HSL, but uses Value (brightness) instead of Lightness, widely used in Photoshop
  • Native support in CSS: hsl() / hsla() functions, perfect for responsive design and theme systems
  • Intuitive for designers: Adjust shade/tint without changing the base color

💼 Industry Applications

UI/UX Design · CSS Theme Development · Digital Art · Photo Editing · Brand Color System · Animation Design

🎯 Pro Learning Tips

  • Always use RGB/Hex/HSL for digital products → CMYK exclusively for printed materials
  • Never convert CMYK directly to RGB for web (severe color distortion will occur)
  • Hex is the most reliable format for cross-browser consistency in web development
  • HSL is the best choice for building theme systems and responsive color palettes
  • Always check color contrast for accessibility (WCAG 2.1 AA/AAA standards)
