RGB Color Model

RGB is composed of Red, Green and Blue. Each channel ranges from 0 to 255. It is the mainstream color mode for electronic screens, web pages and digital images.

CMYK Color Model

CMYK includes Cyan, Magenta, Yellow and Key Black. It is mainly used for physical printing, posters, magazines and paper design.

Hexadecimal Code

Hex code starts with #, using six digits to convert RGB data. It is the most commonly used color writing method in front-end development.
